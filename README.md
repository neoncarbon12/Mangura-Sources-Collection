# Mangura Sources Collection

A community-maintained collection of manga sources for **Mangura**. Import it once to add every source currently available in this collection.

## Import the collection

Copy this URL:

```text
https://neoncarbon12.github.io/Mangura-Sources-Collection/collection.json
```

Then in Mangura:

1. Open **Settings → Sources**.
2. Tap the **+** button.
3. Choose **From URL**.
4. Paste the collection URL and tap **Import**.

You can also import it from the **Search** screen using **+ → From URL**.

## What gets imported

The collection currently contains **3 community sources**. Provider details appear inside Mangura after import rather than being promoted on the collection website.

After importing, enabled sources appear on the Search screen. You can enable, disable, update, or remove them from **Settings → Sources**.

## Keeping sources updated

Pull down on the Sources screen to check installed sources for updates. Mangura remembers each source's update URL, so you do not need to import the collection again for normal source updates.

If new sources are added to this collection later, import the collection URL again. Sources you already have at the same or a newer version will be skipped.

## Having trouble?

- Make sure your device is online and you are using a Mangura version that supports source imports.
- Check that the [collection file](https://neoncarbon12.github.io/Mangura-Sources-Collection/collection.json) opens in your browser.
- If one source stops working, it may need an update after the provider website changes.
- [Report a source issue](https://github.com/neoncarbon12/Mangura-Sources-Collection/issues) and include the source name and the error shown by Mangura.

> These are unofficial, community-maintained source definitions. They are not affiliated with, endorsed by, or supported by the websites they connect to. Mangura and this collection do not host manga content.
